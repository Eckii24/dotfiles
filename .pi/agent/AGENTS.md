# Questions
- If you have open questions, always use the questionary tool to ask them before you start any implementation
- If you still make assumptions, write them down in the implementation and explain why you made those assumptions

# Sub Agents
- Use sub-agents for context management by delegating tasks
- You will mainly orchestrate the tasks, if the task is not simple and can be delegated

# Work progress
- If you (or a sub-agent) create a spec/plan/tasks/..., always write it to a markdown file inside `.ai/` folder
- Update your current progress in the markdown file, so that you can keep track of your work and share it with other agents if needed
